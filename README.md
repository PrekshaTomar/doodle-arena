# 🎨 Doodle Arena

A multiplayer drawing and guessing game like Skribbl.io

## 🚀 Features
- Real-time drawing
- Chat system
- Multiplayer support
- Timer & scoring

## 🛠 Tech Stack
- React
- Node.js
- Socket.io

## ▶️ Run Locally

### Backend
cd backend
node server.js

### Frontend
cd frontend
npm start
